# Подключение к удалённой БД SurrealDB

1. **Скачайте Surrealist**  
   Это графическое приложение для удобной работы с SurrealDB и выполнения запросов.  
   ⚠️ Веб-версия Surrealist **не подключается** к удалённой базе — обязательно скачайте десктопную версию.  
   Ссылка: [https://surrealdb.com/surrealist?download](https://surrealdb.com/surrealist?download) (весит ~30 МБ).

2. **Запустите Surrealist**

3. **Создайте подключение**  
   - На главной странице в разделе **Your Instances** нажмите **Create Connection**.  
   - Заполните поля:
     - **Название**: любое (например, `RemoteDB`)
     - **Протокол**: `WS`
     - **Адрес**: `45.149.234.80:8000`
     - **Тип доступа**: `root`
     - **Логин**: `admin`
     - **Пароль**: Спросить у DevOps

4. **Нажмите Подключиться**

✅ Готово! Вы подключены к удалённой базе данных.
