### US-01  
**Как** участник,  
**я хочу** видеть на главной странице календарь предстоящих соревнований с датами и ссылками на регистрацию,  
**чтобы** не пропускать важные мероприятия и легко переходить к регистрации.

**Критерии приемки:**  
- [ ] Отображается список соревнований с названием, датой, статусом («Регистрация открыта», «Скоро», «Завершено»).  
- [ ] Каждое соревнование содержит прямую ссылку на внешнюю платформу (Яндекс.Контест, Codeforces и др.).  
- [ ] Страница доступна без авторизации.  

---

### US-02  
**Как** менеджер по соревнованиям,  
**я хочу** вручную добавлять и редактировать карточки соревнований через админ-панель,  
**чтобы** централизованно информировать участников.

**Критерии приемки:**  
- [ ] В админке есть форма с полями: название, дата, описание, тип (внешнее/внутреннее), ссылка на регистрацию.  
- [ ] После публикации соревнование сразу появляется в календаре.  
- [ ] Система не запрашивает персональные данные участников.  

---

### US-03  
**Как** участник,  
**я хочу** видеть архив прошедших соревнований с результатами (если доступны),  
**чтобы** анализировать свой прогресс и изучать задачи.

**Критерии приемки:**  
- [ ] На отдельной странице отображается список завершённых соревнований.  
- [ ] Для каждого соревнования указаны: дата, название, внешняя платформа.  
- [ ] Если результаты загружены — отображается ссылка на scoreboard или таблицу.  

---

### US-04  
**Как** участник,  
**я хочу** получать уведомления о новых соревнованиях и обновлениях,  
**чтобы** быть в курсе событий БЦСП.

**Критерии приемки:**  
- [ ] В профиле можно включить in-app уведомления.  
- [ ] При публикации нового соревнования или новости система автоматически оповещает подписанных пользователей.  
- [ ] Email-рассылка — опционально, не обязательна для MVP.  

---

### US-05  
**Как** участник,  
**я хочу** видеть в личном кабинете историю своих участий в соревнованиях,  
**чтобы** отслеживать свою активность и готовить портфолио.

**Критерии приемки:**  
- [ ] В профиле отображается список соревнований, в которых пользователь участвовал.  
- [ ] Для каждого соревнования указаны: название, дата, внешняя платформа.  
- [ ] Результаты (место, задачи) отображаются, если они были добавлены администратором.  

---

### US-06  
**Как** участник,  
**я хочу** иметь возможность вручную отметить факт участия в соревновании,  
**чтобы** система могла фиксировать мою историю даже без автоматической интеграции.

**Критерии приемки:**  
- [ ] В карточке соревнования есть кнопка «Я участвовал».  
- [ ] После подтверждения событие добавляется в историю профиля.  
- [ ] Участник может указать своё место и количество решённых задач (опционально).  

---

### US-07  
**Как** участник,  
**я хочу** видеть общий рейтинг участников с возможностью фильтрации по категориям (школьники, студенты, ветераны),  
**чтобы** оценивать свой уровень и находить мотивацию.

**Критерии приемки:**  
- [ ] На отдельной странице отображается рейтинг с сортировкой по баллам.  
- [ ] Присутствуют фильтры по категориям участников.  
- [ ] Для MVP рейтинг рассчитывается по упрощённой формуле (участия + победы).  

---

### US-08  
**Как** участник,  
**я хочу** иметь возможность сгенерировать PDF-версию своего портфолио с достижениями,  
**чтобы** использовать его при поступлении или трудоустройстве.

**Критерии приемки:**  
- [ ] В профиле есть кнопка «Создать портфолио».  
- [ ] PDF содержит: список соревнований, результаты, общую статистику.  
- [ ] Файл имеет чистое и профессиональное оформление.

### US-09  
**Как** менеджер по соревнованиям,  
**я хочу** публиковать новости и анонсы через админ-панель,  
**чтобы** информировать сообщество без технических сложностей.

**Критерии приемки:**  
- [ ] В админке есть форма для создания новости (заголовок, текст, теги).  
- [ ] После публикации новость отображается в общем списке.  
- [ ] Новости доступны без авторизации.  

---

### US-10  
**Как** гость,  
**я хочу** видеть новости и анонсы без регистрации,  
**чтобы** понять, насколько активен БЦСП, прежде чем присоединиться.

**Критерии приемки:**  
- [ ] Страница `/news` открыта для всех.  
- [ ] Все посты отображаются в хронологическом порядке.  

---

### US-11  
**Как** организатор внутреннего соревнования БЦСП,  
**я хочу** видеть список участников, зарегистрированных на моё мероприятие,  
**чтобы** эффективно организовывать процесс.

**Критерии приемки:**  
- [ ] Для внутренних соревнований БЦСП в админке отображается список участников.  
- [ ] Список можно экспортировать в CSV.  
- [ ] Система не хранит паспортные данные — только nickname и email.  

---

### US-12  
**Как** участник,  
**я хочу** иметь возможность зарегистрироваться на внутренние соревнования БЦСП одним кликом,  
**чтобы** не заполнять формы каждый раз.

**Критерии приемки:**  
- [ ] В карточке внутреннего соревнования есть кнопка «Зарегистрироваться».  
- [ ] После нажатия пользователь добавляется в список участников.  
- [ ] Регистрация требует авторизации на платформе.  

---

### US-13  
**Как** наблюдатель (тренер, работодатель, родитель),  
**я хочу** видеть публичные профили участников с достижениями, но без личных данных,  
**чтобы** оценивать уровень подготовки.

**Критерии приемки:**  
- [ ] Профиль доступен по прямой ссылке без авторизации.  
- [ ] Отображаются: никнейм, категория (школьник/студент/ветеран), история соревнований.  
- [ ] Реальное имя, email и другие приватные данные скрыты.  

---

### US-14  
**Как** менеджер по соревнованиям,  
**я хочу** иметь возможность вручную вносить результаты соревнований через простой интерфейс,  
**чтобы** обновлять рейтинги без технических сложностей.

**Критерии приемки:**  
- [ ] В админке есть раздел для внесения результатов.  
- [ ] Поддерживается ручной ввод или импорт из CSV.  
- [ ] После сохранения рейтинги участников обновляются автоматически.  

---

### US-15  
**Как** участник,  
**я хочу** видеть различие между внутренними мероприятиями БЦСП и внешними соревнованиями,  
**чтобы** понимать, где возможна упрощённая регистрация, а где — только внешние ссылки.

**Критерии приемки:**  
- [ ] В календаре соревнования помечены как «Внешнее» или «БЦСП».  
- [ ] Для БЦСП отображается кнопка «Зарегистрироваться», для внешних — «Перейти к регистрации».  
- [ ] Иконки или цветовая индикация помогают быстро различать типы.
