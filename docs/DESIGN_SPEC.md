## Типографика

### Шрифты

| Шрифт | Назначение |
|-------|-----------|
| **Geist** (sans-serif) | Основной шрифт — весь UI |
| **Geist Mono** (monospace) | Код,验证码, технические данные |
| **Arial, Helvetica, sans-serif** | Фоллбэк |

### Шкала размеров

| Размер | Где используется |
|--------|-----------------|
| **48px** | Логотип «БЦСП» в хедере (десктоп) |
| **40px** | Заголовки секций (приветствие, события) |
| **36px** | Логотип «БЦСП» (планшет, ≤1300px) |
| **32px** | Навигация хедера (десктоп) |
| **24px** | Заголовки страниц авторизации, карточки |
| **22px** | Текст подписки на Telegram |
| **20px** | Навигация хедера (≤1300px) |
| **16px** | Основной текст, инпуты авторизации |
| **15px** | Мелкий текст, кнопка «Зарегистрироваться» |
| **14px** | Лейблы форм, плейсхолдеры, текст форм |
| **12px** | Текст в чате, мелкий UI |

> **Важно:** `html { font-size: 10px; }` в модулях авторизации, поэтому `1rem = 10px`. На остальном сайте `1rem = 16px` (стандарт).

## Layout-паттерны

### Общая структура страницы

```tsx
<div style={{ display: 'flex', flexDirection: 'column', minHeight: '100vh' }}>
  <Header />              {/* фиксированный */}
  <main style={{ flex: '1 0 auto' }}>
    {children}            {/* контент страницы */}
  </main>
  <Footer />

```

### Flexbox

| Паттерн | Где |
|---------|-----|
| `flex; justify-content: center; align-items: center` | Центрирование (auth-страницы, кнопки) |
| `flex; justify-content: space-between; align-items: center` | Заголовки карточек, календарь, профиль |
| `flex; flex-direction: column` | Формы, чат, модалки |
| `flex; flex-wrap: wrap` | Фильтры, список событий |
| `flex; gap: N` | Чат, фильтры, профиль |

### Grid

| Паттерн | Где |
|---------|-----|
| `grid: 1fr → md: 1fr 1fr` | Dashboard (responsive) |
| `grid: auto-fill, minmax(250px, 1fr)` | Профиль: система-линки |
| `grid: repeat(2, 1fr)` | Профиль: форма редактирования |
| `grid: auto-fit, minmax(150px, 1fr)` | AtCoder stats |
| `grid: 1 → md: 3` | Footer (Tailwind) |
